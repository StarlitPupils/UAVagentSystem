# 📋 UAVagent 更新日志

## [1.2.0] - 2025-01-XX

### 🚀 重大升级

#### 检测优化
- ✅ 5模型集成融合 (YOLOv11x + YOLOv8x + YOLOv11n + YOLOv10n + RT-DETR-l)
- ✅ CLAHE 自适应图像预处理（低光照/雾霾增强）
- ✅ 自适应检测阈值 + 质量门控 + 异常模型熔断
- ✅ 灰度图自动检测（合成场景排除RT-DETR）
- 📊 VisDrone 100帧: MOTA 0.825 (+18.5pp vs 单模型)

#### 跟踪优化
- ✅ EKF 扩展卡尔曼滤波（8状态）
- ✅ HSV 直方图外观匹配（64维统一）
- ✅ 轨迹插值修复断裂
- ✅ 深度ReID特征提取（OSNet备选，自动降级）
- 📊 ID Switches: 0 (完美保持)

#### LLM 推理优化
- ✅ Ollama 本地模型支持（离线推理）
- ✅ LLM 路由（简单任务本地，复杂任务云端）
- ✅ 多轮 ReAct 推理（Thought→Action→Observation）
- ✅ 三级降级策略完善

#### 工程化
- ✅ CI/CD 流水线（GitHub Actions）
- ✅ 开源打包标准化（LICENSE, CHANGELOG, .gitignore）
- ✅ 虚拟环境自动配置
